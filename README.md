# МастерGO

MVP сервиса быстрых бытовых услуг без комиссии. Пользователь публикует задачу,
указывает цену, время и район, после чего заказ получают свободные мастера нужной
категории. Первый подходящий отклик можно принять сразу или обсудить условия в чате.

## Что есть в прототипе

- кабинет заказчика и адаптивная мобильная версия;
- категории услуг: сантехника, мебель, питомцы, фитнес и помощь по дому;
- создание заказа в три шага с фото/видео, адресом, датой и ценой;
- имитация рассылки доступным мастерам поблизости;
- список откликов с рейтингом, сроком прибытия и встречной ценой;
- выбор мастера, чат, статус выполнения и обязательный двусторонний отзыв;
- подписочная модель и акцент на отсутствии комиссии.

## Запуск

```bash
npm install
npm run dev
```

Production-сборка:

```bash
npm run build
```

## Границы MVP

Сейчас это интерактивный frontend-прототип с локальным состоянием и демо-данными.
Для пилота нужны авторизация, backend/API, база данных, объектное хранилище,
геокодинг и карта, realtime-чат и push-уведомления, платежи подписки, модерация и
раздельные кабинеты заказчика и мастера.

## Приоритет следующего этапа

1. Авторизация по телефону и выбор роли.
2. Создание и хранение заказов с геопоиском мастеров.
3. Realtime-отклики, торг и блокировка заказа после принятия.
4. Подписка, проверка мастеров и двусторонние отзывы.
5. Мобильные push-уведомления и аналитика скорости закрытия заказа.
